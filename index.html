Learn Spanish

Learn Spanish is a Console application made to practice Spanish words. You are able to learn the words from Dutch to Spanish and from Spanish to Dutch. An output will be given when a word is written correct or incorrect. When it is incorrect, the program shows the user what answer was right. The program also checks on accent marks. In that way, the user can practice until he or she knows the words sufficient.

The application is implemented in such a way that it is very easy to expand the library with more words. The lists of words are saved in JSON files which are imported in the program.

Motivation

I like to learn new languages and had a great time on high school with English and German lessons. Learning Spanish was already on my bucketlist when I found out that Hogeschool Rotterdam has an elective course "Spanish for Beginners". I came with the idea to make this program during the course. Of course I could have used a learning program from the internet which already exist, but I liked to challenge myself to make a own small program. With this program I have managed to know the Spanish words and numbers sufficient for the exam.
